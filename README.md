# FedAvg-Modeltraining-
This project implements a federated learning framework for transportation mode classification using mobility datasets collected from multiple cities. 
In Part 1, a FedAvg-based federated multilayer perceptron model is trained across distributed clients to classify common transportation modes (car vs walk) while preserving data privacy.
In Part 2, the project addresses heterogeneous data challenges by focusing on the KAUST dataset, which contains additional transportation modes. Local multi-class models, client-level simulations, federated ensemble strategies, and non-IID experiments are implemented to study how federated learning can be extended to handle real-world mobility datasets with differing label spaces.
